# Exploratory Data Analysis (EDA)

## Objective
To analyze and understand the dataset by exploring patterns, trends, and relationships.

## Tasks Performed
* Asked meaningful questions about the dataset
* Explored structure (columns, data types)
* Performed statistical analysis
* Identified trends and patterns
* Detected anomalies and missing values

## Tools Used
* Python
* Pandas
* Matplotlib
* Seaborn

## Key Insights
* Found relationships between variables
* Identified data distribution patterns
* Detected outliers and missing data
